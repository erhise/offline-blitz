# Instructions
1. clone repo
2. install dependencies
3. run script

package.json:
```
scripts: {
  "run": // run environment in terminal for console log (run app.ts)
}
dependencies: {
  "rxjs",
  "typescript",
}
```

structure:
```
root
  src/
    app.ts <-- main file
```

helpers:
* Module with common rxjs import
  * e.g. include common imports directly in app.ts
